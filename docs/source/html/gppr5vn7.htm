Syntax errors

If your source code contains syntax errors, GpProfile may not be able to properly instrument the target. If this happens, you may not receive an error message indicating a syntax problem until GpProfile invokes the Delphi IDE, and you attempt to execute your application. GpProfile is not designed to detect syntax errors. The best recommendation is to make sure that your project will compile under the Delphi IDE prior to running it through GpProfile.
